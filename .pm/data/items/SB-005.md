## Motivation

Citizen Sleeper works because the die is placed into a specific action slot with visible stakes. Lifelines should make the player choose what kind of phone practice today’s capacity is spent on.

## Scope

Each practice path slot defines `safe`, `risky`, or `locked`; requirement if locked; effect on room permission/progress; and pressure side effect. Resolver should receive `activityClockId` and `slotType`, not only `dieFace`.

## Acceptance

- Player chooses a specific practice path + slot before trying.
- Safe and risky slots produce different evidence/pressure effects.
- Locked slots explain what is missing in plain language.
