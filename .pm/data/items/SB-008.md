## Motivation

The state-object table from the Cultist note is strong, but Lifelines should not surface abstract mental labels as primary nouns. A state object should first be a concrete thing in the apartment or on the case desk; Frank’s interpretation comes second.

## Scope

Extend or replace `PressureObject` fields so each object has concrete `objectLabel`, `visibleState`, `roomAnchor`, and `functionalRead`. Rewrite at least five pressure objects to lead with physical things.

## Acceptance

- UI cards lead with object names, not abstract state names.
- Overlay labels attach to concrete props, not floating mental-state tags.
- At least five pressure objects are rewritten as physical apartment/table things.
