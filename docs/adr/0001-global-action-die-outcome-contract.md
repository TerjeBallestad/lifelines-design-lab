# Global action die outcome contract

Action die faces set the global visible odds for negative, neutral, and positive action outcomes across Lifelines action contexts. A `6` guarantees a positive outcome; a `5` is split between positive and neutral; `4-3` can land positive, neutral, or negative; and `2-1` is split between neutral and negative. Context decides what those outcome classes mean, what evidence appears, and which trust, friction, clocks, or board states move, but it does not replace the die face with an opaque readiness formula.
